Retrouvez les fonctionnalités d'Ammoniaque.io directement dans votre interface d'administration Wordpress

== Description ==
Ammoniaque.io est un service de veille orthographique qui permet aux webmasters d'obtenir une relecture et correction de toutes les pages publiques de leur site, anciennes et nouvelles, par des personnes qualifiées plutôt que par un simple logiciel de correction automatique. Dès lors qu'une erreur est rencontrée, une alerte est envoyée par email.

Le webmaster a également accès à une interface lui permettant de réaliser en toute confidentialité et en illimité la relecture -en quelques heures si nécessaire- d'un écrit avant sa publication. Tout écrit en lien avec son site peut être traité : service client, marketing ou encore futures nouvelles pages.

Cette extension permet de retrouver toutes les fonctionnalités d'un compte Ammoniaque.io directement depuis l'espace administration de son site WordPress.

== Installation ==
Pour que le plugin soit actif il est nécessaire d'être inscrit sur le site Ammoniaque.io
